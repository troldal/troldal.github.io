OpenXLSX::XLTokenizer Class Reference

The XLTokenizer class is a convenience class for tokenizing a string, i.e. splitting it up in tokens, separated by one or more defined separators. It is based on the Tokenizer class developed by Song Ho Ahn (http://www.songho.ca/misc/tokenizer/tokenizer.html) More...
